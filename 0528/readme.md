### fork()는 한 번 호출되면 두 번 리턴한다
- 자식 프로세스에게는 0리턴
- 부모 프로세스에게는 자식 프로세스 ID를 리턴
- 부모 프로세스와 자식 프로세스는 병행적으로 각각 실행을 계속함
